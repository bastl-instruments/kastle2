# Kastle 2 FX Wizard

Experimental effects firmware for the Kastle 2 platform with 9 unique effects. Has three main parameters: Time, Feedback and Amount (Dry & Wet and more).  

## FX Wizard File Format

You can use a specific format to load your custom rhythms into the FX Wizard. See the [FX_WIZARD_FILE_FORMAT.md](FX_WIZARD_FILE_FORMAT.md) for that or just use [FX Wizard Chamber](http://apps.bastl-instruments.com/fx-wizard-chamber) to get the current firmware with custom rhythms.
